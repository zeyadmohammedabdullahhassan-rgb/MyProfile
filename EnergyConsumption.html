Energy Consumption Data Analysis

End-to-End Data analysis Project using Power BI

01 Step 1: Reviewing the Available Data in Power BI

In the first step of the analysis, I reviewed the available datasets in Power BI to understand the structure and relationships between the tables. The analysis file contains three main tables.

Table 1 – Building Master

This table contains the master data for 11 different buildings. It includes the essential building information used to link consumption and cost data across the model.

02

Table 2 – Energy Consumptions

This table presents the energy consumption units for each building on a yearly basis from 2016 to 2019. It provides detailed insights into usage patterns across different buildings over time.

03

Table 3 – Rates

This table displays the Unit Cost for each year along with the corresponding Energy Type (Water, Gas, Electricity). It is used to calculate the total energy cost by multiplying consumption units by the unit cost.

04 Step 2: Transforming the Data using Power Query

In this step, we transformed the data in Power Query to prepare it for analysis. Specifically, we added a unique Id column in both the Rates and Energy Consumptions tables by combining the Year and Energy Type columns. This allows us to create a proper relationship between the two tables in the data model.

Creating the Unique Id Column

The unique Id helps link the datasets accurately. For example, combining Year = 2025 and Energy Type = Electricity results in 2025_Electricity. Similarly, Year = 2025 and Energy Type = Gas results in 2025_Gas.

This approach ensures that each row is uniquely identified, enabling reliable relationships and analysis across tables.

05 Step 3: Data Modeling

In this step, we focused on data modeling to establish relationships between the tables in the Power BI data model. By using the Unique Id columns created in the previous step, we connected the Rates table with the Energy Consumptions table.

Building Relationships

Establishing proper relationships ensures that the data can be analyzed correctly across tables. For example, linking the Unique Id columns allows us to match the consumption of a specific energy type in a given year with its corresponding rate.

After setting up the relationships, we validated the model to make sure all tables interact correctly, providing accurate results in our reports and visualizations.

06 Step 4: Data Visualization and Dashboard Making

In this step, we created dashboards to visualize the data and gain insights. The Overview section shows key metrics such as:

  • Count of Buildings
  • Total Cost
  • Units Consumed
  • Energy Type Breakdown

These visuals help quickly understand the overall energy consumption and costs across all buildings.

Insights & Recommendations

Key Insights

  • Water Consumption: Highest usage (88.49% of total energy), but cost remains relatively stable. Peak in 2017, slight increase in 2019. Opportunity for conservation efforts without affecting budget.
  • Gas Consumption: Lowest usage, decreasing trend from 2016–2019, reflected in Gas Cost. Likely due to efficiency improvements and shift to alternative energy sources.
  • Need for Deeper Analysis: More details (building type, activity, policy changes, infrastructure, operations) are needed to fully explain water and gas consumption patterns.
  • Additional Insights: Analysis by city, time of day, or day of week can reveal peak usage and areas for conservation, improving resource management.

Recommendations

  • Conservation Strategies: Implement water-saving and energy-reduction initiatives based on dashboard insights.
  • Efficiency Measures: Investigate reasons for gas reduction and explore further operational efficiency opportunities.
  • Enhanced Data Analysis: Include building type, location, and time periods to better understand energy patterns.
  • Continuous Monitoring: Use dashboards to track changes, detect anomalies, and make data-driven decisions for cost optimization and effective resource management.

Overall, leveraging Power BI and data visualization provides actionable insights that support sustainable energy practices, cost reduction, and operational efficiency.
